You are a senior full-stack software engineer and DevOps engineer.

Your task is to create a **minimal but production-quality prototype** of an **AI Technology News Search Agent system**.

IMPORTANT CONSTRAINT:

The entire project MUST run inside a **Docker development environment**.

Never install any dependencies on the host machine.

All dependencies must be installed inside Docker containers.

The project must run with:

docker compose up

and be accessible at:

http://localhost:8000

---

# Goal

Create a web application where a user can use an AI agent to search for the latest AI-related technical news and research papers.

The system must support **two types of search**:

1. Automatic search for latest AI news
2. User-driven search using keywords or descriptions

Both types of search must use the same AI agent pipeline.

Button:

3. summarize results using the LLM
4. store them in SQLite
5. display them in the UI

# Agent Architecture

Implement a class:

AIResearchAgent

Responsibilities:

search_sources(query)

Search external sources using the query.

If query is None:

use default automatic queries.

summarize_article()

Use OpenAI API to summarize content.

save_article()

Save article metadata and summary to SQLite.

run(query=None)

Main orchestration function.
